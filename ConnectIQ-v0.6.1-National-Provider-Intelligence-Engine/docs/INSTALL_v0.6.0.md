# Installation
1. Extract the ZIP to a normal folder outside `C:\connectiqvscode`.
2. Double-click `install-connectiq.bat`.
3. The installer backs up the current installation, preserves environment files, copies the release, runs `npm ci`, lint, 100 tests, and the production build.
4. Run `verify-install.bat` at any time.
5. Run `rollback.bat` to restore the most recent installer backup.
