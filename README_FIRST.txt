ConnectIQ v0.7.6 — OpenAI-Only Provider Architecture

1. Close all ConnectIQ frontend and backend terminals.
2. Extract this ZIP to a normal folder. Do not run it from the ZIP preview.
3. Double-click install-connectiq.bat.
4. Restart the backend from C:\connectiqvscode\functions with npm run dev.
5. Restart the frontend from C:\connectiqvscode with npm run dev.
6. Test the customer and admin provider lookup.

The installer preserves both .env files and your OpenAI API key.
