# Install ConnectIQ v0.4.0

1. Stop the frontend and backend with Ctrl+C.
2. Back up the current project folder.
3. Copy the existing root `.env` and `functions/.env` into this release.
4. Run `install-connectiq-v0.4.0.bat` on Windows.
5. Start the backend from `functions` with `npm run dev`.
6. Start the frontend from the project root with `npm run dev`.
7. Open `http://localhost:5173/internet`.
